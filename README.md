# Tic Tac Toe Game (C++)

This project is a **console-based Tic Tac Toe game written in C++**. It was developed as my **1st semester Programming Fundamentals project**. The game allows two players to play Tic Tac Toe on a 3×3 grid in the terminal.

## Features

* Two-player gameplay (Player X and Player O)
* Simple and interactive 3×3 board
* Help menu explaining the rules
* Input validation for user entries
* Score tracking system
* Player records, including wins, points, and ties
* Option to play the game again

## How to Play

1. The game is played on a **3×3 board**.
2. Player 1 uses **X** and Player 2 uses **O**.
3. Players take turns entering a number from **1 to 9** to choose a position on the board.
4. The first player to get **three marks in a row (horizontal, vertical, or diagonal)** wins the game.
5. If all spaces are filled and no player wins, the game ends in a **draw**.

## Scoring System

* Regular win: **5 points**
* Diagonal win: **10 points**
* Draw: **0 points**
